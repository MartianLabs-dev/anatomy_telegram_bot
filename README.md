# Telegram Bot: Анатомия и Биомеханика 🏋️‍♂️

Полностью функциональный Telegram-бот для тренировки знаний по фитнес-анатомии. Бот работает в режиме "Флэш-карточек", позволяя тянуть случайные билеты, смотреть подсказки и вести статистику.

## Выбранный стек технологий
*   **Python 3.10+**
*   **aiogram 3.x** — самый современный, быстрый и актуальный асинхронный фреймворк для Telegram ботов.
*   **aiosqlite** — легковесная локальная база данных для сбора статистики без сложных настроек.
*   **Хостинг: PythonAnywhere** — лучший выбор для бесплатных ботов с локальной базой данных SQLite. В отличие от других бесплатных хостингов (Render/Railway), здесь файловая система не удаляется при перезагрузке сервера, поэтому ваша статистика в БД никогда не потеряется.

## Функционал
1.  **Тянуть билет**: 5 случайных вопросов из базы в 200 билетов.
2.  **Подсказки**: Показывает начало ответа. Использование подсказки сохраняется в статистику.
3.  **Самопроверка**: Кнопки "Знал" / "Ошибся" после ответа.
4.  **Подробная статистика**: Подсчет пройденных билетов, идеальных билетов (без ошибок), правильно/неправильно отвеченных вопросов.

## 🚀 Инструкция по запуску и Деплою на PythonAnywhere

### Шаг 1. Получение токена Telegram
1. Зайдите в Telegram и найдите бота **@BotFather**.
2. Напишите команду `/newbot`.
3. Придумайте имя и username (с окончанием _bot).
4. Скопируйте полученный **API Token** (например, `12345:ABCdef...`).

### Шаг 2. Развертывание на PythonAnywhere (Бесплатно)
1. Зарегистрируйтесь на сайте [pythonanywhere.com](https://www.pythonanywhere.com/).
2. Перейдите во вкладку **Files** и создайте новую папку `telegram_bot`.
3. Загрузите туда все 5 файлов из нашей папки: `bot.py`, `db.py`, `parser.py`, `questions.json`, `requirements.txt`.
4. Перейдите во вкладку **Consoles** и откройте **Bash**.
5. Напишите команды для установки библиотек:
   ```bash
   cd telegram_bot
   pip3 install -r requirements.txt
   ```
6. В этой же консоли пропишите ваш Токен в переменные среды и запустите бота:
   ```bash
   export BOT_TOKEN="ВАШ_ТОКЕН_ОТ_БОТ_ФАЗЕРА"
   python3 bot.py
   ```
7. *Дополнительно (Чтобы бот работал всегда):* На бесплатном тарифе консоль может иногда засыпать. Перейдите во вкладку **Tasks** (Always-on tasks) и укажите команду запуска там (если Always-on доступен), либо просто запускайте его через Bash. Для полностью бесплатных Always-On задач часто используют сервис **Render.com**, но там придется перевести базу данных с SQLite на PostgreSQL (чтобы статистика не сбрасывалась).

Бот полностью готов к работе!
